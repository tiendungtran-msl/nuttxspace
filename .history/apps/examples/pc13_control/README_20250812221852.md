# PC13 Control Application

Ứng dụng NuttX để điều khiển chân PC13 trên STM32F411 BlackPill.

## Tính năng

- Điều khiển GPIO PC13 (LED onboard)
- Chế độ thủ công: on/off/toggle
- Chế độ tự động: nhấp nháy với khoảng thời gian có thể cấu hình
- Giao diện dòng lệnh tương tác
- Cấu trúc modular, dễ mở rộng

## Cách sử dụng

### Cấu hình Kconfig

```bash
make menuconfig
# Application Configuration -> Examples -> PC13 Control Example
